# Projection Mapping Lab

Webカメラで検出した物理オブジェクトを Matter.js の静的コリジョンとして扱うリアルタイム・プロジェクションマッピング実験アプリです。

OpenCV.js で HSV 色検出を行い、マーカーの輪郭を投影キャンバス座標へ変換して、ボールが跳ね返る物理デモとして表示します。

---

## セットアップ・起動

```bash
npm install
npm run dev -- --port 5173
```

```
http://localhost:5173/
```

---

## 画面構成

| エリア | 内容 |
|--------|------|
| 左パネル（コントロール） | ボタン・スライダー群 |
| 右上（Projector Output） | 投影キャンバスのプレビュー（1280×720） |
| 右下（Camera） | Webカメラ映像のプレビュー（640×480） |

## ステータス表示

パネル上部に現在の状態を表示します。

| 項目 | 内容 |
|------|------|
| FPS | レンダリングフレームレート |
| 検出数 | 現在検出中のマーカー数 |
| 補正点 | 設定済みの4点補正点数（n/4） |
| 入力 | `CAM`（カメラ）/ `DEMO`（デモ）/ `OFF` |

## ボタン一覧

### 制御

| ボタン | 動作 |
|--------|------|
| 停止 / 再開 | 物理演算・マーカー認識ループの一時停止と再開 |
| カメラ / デモ | デモモードはダミーマーカーで動作確認できる。カメラ不要 |
| プロジェクターへ出力 | 別ウィンドウを開いてプロジェクター用出力を表示する |
| 左右反転 | 出力ウィンドウの映像を左右に反転する（トグル） |
| 上下反転 | 出力ウィンドウの映像を上下に反転する（トグル） |
| テストパターン | カラーバー＋チェッカーボードをプレビューと出力ウィンドウに表示する（トグル） |

### ボール・コリジョン

| ボタン | 動作 |
|--------|------|
| 自動スポーン / スポーン停止 | ボール自動追加の開始・停止。物理演算停止中はスポーンも止まる |
| 4点補正 | カメラ映像の4隅をクリックしてホモグラフィ変換を設定するモードに入る |
| 補正初期化 | 4点補正をデフォルト値に戻す |
| スポイトで色を選択 | カメラ映像をクリックしてマーカー検出色を自動取得する |
| クリックでボール配置 | 投影プレビューをクリックした位置にボールを追加する（トグル） |
| クリックでコリジョン配置 | 投影プレビューをクリックした位置に静的コリジョンを追加する（トグル） |
| コリジョン消去 | 手動追加したコリジョンをすべて削除する |

## コントロールパネル（折りたたみ可能セクション）

タイトル行をクリックして各セクションを折りたたみ・展開できます。

---

### 認識モード

| 項目 | 内容 |
|------|------|
| 全体認識 | カメラ4点 → Canvasの4隅（1280×720）にホモグラフィ変換 |
| 部分認識 | カメラ4点 → Interaction Areaの4点にホモグラフィ変換 |
| 保存 | 認識モード・補正点・Interaction Area・投影オフセット・エリア枠表示をJSONに保存 |
| 読み込み | 保存したJSONを読み込んで設定を復元 |

---

### Interaction Area（部分認識モード選択時のみ表示）

物理演算とマーカー認識が有効になる投影キャンバス上の領域です。  
台形など**矩形以外の任意の四角形**を設定できます。

| 項目 | 内容 |
|------|------|
| 4点設定 | 投影プレビューを4点順番にクリックして形状を確定する |
| 初期化 | デフォルト矩形（左上160,90 ／ 右下1120,630）に戻す |
| エリア枠を表示 | 投影プレビューと出力ウィンドウへのオレンジ枠描画を切り替える |

---

### 色検出

マーカーとして検出するHSV色の範囲を設定します。

| スライダー | 範囲 | デフォルト |
|------------|------|-----------|
| Hue（色相） | 0〜179 | 150 |
| 許容幅 | 4〜45 | 14 |
| 彩度（下限） | 0〜255 | 90 |
| 明度（下限） | 0〜255 | 80 |

- 「検出マスク」プレビューで白い部分が検出対象領域を示します。
- 「保存」「読み込み」で色設定をJSONファイルに書き出し・復元できます。
- 「初期値に戻す」でデフォルト値に一括リセットできます。

---

### 投影オフセット

ホモグラフィ変換後の残ズレを手動で微調整します。

| スライダー | 範囲 | デフォルト |
|------------|------|-----------|
| X | −160〜160 | 0 |
| Y | −120〜120 | 0 |

---

### カメラオーバーレイ

Projector Output プレビューにWebカメラ映像を半透明で重ねて表示します。  
4点補正で定義した領域の内側のみ描画されます。出力ウィンドウには表示されません。

| スライダー | 範囲 | デフォルト |
|------------|------|-----------|
| 不透明度 | 0.00〜1.00（0.05刻み） | 0.40 |

---

### ボール自動追加

指定した座標に一定間隔でボールを自動スポーンします。  
「自動スポーン」ボタンで開始し、「スポーン停止」で止めます。

| スライダー | 範囲 | デフォルト |
|------------|------|-----------|
| X（スポーン位置） | 0〜1280 | 640 |
| Y（スポーン位置） | 0〜720 | 40 |
| 間隔（秒） | 0.5〜30（0.5刻み） | 3.0 |
